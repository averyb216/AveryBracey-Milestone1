IT 3203 - Project Milestone 1

Hypertext Transfer Protocol (HTTP)

My research site exploring HTTP and its role in technology, history, current status, and future directions.

The World Wide Web is one of the most society-changing inventions of the 20th century. It has changed how people share information, collaborate, and do business globally. This site summarizes research on HTTP, the core application-layer protocol that allows browsers and servers to communicate. It covers what HTTP does, how it has evolved, and the direction it is going looking to the future.

Key Terms

HTTP
Protocol for web client–server communication
HTTPS
HTTP over TLS encryption
Request/Response
Client asks and server answers
Stateless
Each request handled independently
QUIC
A modern transport layer network protocol, designed to reduce latency
W3C / IETF
Standards and ethical rules for the web
