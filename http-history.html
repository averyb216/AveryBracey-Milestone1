History of HTTP

HTTP dates back to the late 1980s, and has grown through several versions to improve capability and performance through the years.

Version Year Key Features
HTTP/0.9 1991 Simple, a single line and one method (GET), no headers, HTML could be transmitted
HTTP/1.0 1996 Headers, images & other media, expected look for HTML today
HTTP/1.1 1997 Persistent connections, adapted by many websites for a long period of time, better caching
HTTP/2 2015 Multiplexing, more complex, server push
HTTP/3 2022 Built on QUIC over UDP, connection migration, major companies use it
